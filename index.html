JavaScript Functions: Regex

functions documented

matchLiteral - This will check if pattern is in string with a literal match.

matchSingleCharacterWithMultiplePossibilities - This will find all the characters in possibilities in string.

matchSingleCharactersNotSpecified - This will find all the characters not in possibilities in string.

matchCharactersThatOccurOneOrMoreTimes - This will find matches when the given letter occurs one or more times in string.

findCriminals - This will find one or more criminals represented by letter criminal within a group of other people.

matchBeginningStringPattern - This will find target only in the beginning of the string.

matchEndingStringPattern - This will find target only at the ending of the string.

countAlphanumeric - This will count the number of alphanumeric characters in the given string.
